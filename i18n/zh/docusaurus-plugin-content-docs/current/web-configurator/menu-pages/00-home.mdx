# 网页配置器 - 主页

您可以在此查看设备的系统状态。

-----

## 版本

`Device Name` ( 设备名称 ) - 这将告诉您设备的名称以及上传到设备上的 UF2 文件的名称。
`Current` ( 当前版本 ) - 设备上正在运行的固件的当前版本。
`Latest` ( 最新版本 ) - 在我们的发布区中可用的最新固件版本。

-----

## 闪存

本节显示了有关闪存模块的各种状态。

:::note

GP2040-CE 要求设备至少具有 2MB 闪存（这是树莓派 Pico 的标准配置）。

有少量报告称，来自淘宝（AliExpress）等地方的一些低成本仿制设备只配备了 1MB 闪存。

应避免使用只有 1MB 闪存的设备，因为它们可能无法工作或无法按预期工作。

如果您正在制作自己的设备，请确保使用至少 2MB 的闪存模块。

:::
