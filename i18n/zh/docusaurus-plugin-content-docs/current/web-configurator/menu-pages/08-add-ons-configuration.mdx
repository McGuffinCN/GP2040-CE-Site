# 插件配置

本节用于启用自定义插件，以扩展 GP2040-CE 的功能。由于社区创建了大量插件，每个插件都有其单独的文档页面。

:::caution

部分插件为实验性功能，并非所有插件都能与其他插件互相兼容。

:::

- `玩家编号` - 已由主页面 (设置)[../../web-configurator/menu-pages/01-settings.mdx] 中设置 PID/VID 的功能取代。
- `PS4 模式` - 已移至 (设置)[../../web-configurator/menu-pages/01-settings.mdx] 页面，在 `当前输入模式` 下的 PS4。
- `PS 直通模式` - 已移至 (设置)[../../web-configurator/menu-pages/01-settings.mdx] 页面，在 `当前输入模式` 下的 PS4。
- `Xbox One 直通模式` - 已移至 (设置)[../../web-configurator/menu-pages/01-settings.mdx] 页面，在 `当前输入模式` 下的 Xbox One。
