# 介绍

您的控制器内置了一个电路板，用于处理您的输入并将其转换为计算机或游戏主机能够理解的命令。GP2040-CE 是促进此过程的固件，提供对多个主机和输入模式的兼容性。此外，通过网络配置器和社区开发的插件系统，还有许多其他有用的功能。

不确定您的电路板是否能运行 GP2040-CE？只要它有 RP2040 微处理器，通常就可以。如果您当前的控制器板没有，还有大量社区提供的支持 GP2040-CE 的爱好者板。
