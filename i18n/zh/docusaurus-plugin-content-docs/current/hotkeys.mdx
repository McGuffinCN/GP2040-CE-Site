# 快捷键指南

选择按钮标签以在使用指南中显示：

<InputLabelSelector />
<br />

:::note

所有以下快捷键绑定都可以在网页配置器的 [快捷键设置](./web-configurator/menu-pages/01-settings.mdx#快捷键设置) 中更改或分配为不同的按钮组合。

:::

## 无

此快捷键无功能。使用这个快捷键来禁用特定的快捷键组合。

## 十字键模式

此快捷键将方向模式更改为读取为十字键上的数字输入。

**默认**: <Hotkey buttons={["S1", "S2", "Down"]}/>

## 左摇杆模式

此快捷键将方向模式更改为读取为左摇杆上的输入。方向输入始终是该轴上的最小值、最大值或零值。

**默认**: <Hotkey buttons={["S1", "S2", "Left"]}/>

## 右摇杆模式

此快捷键将方向模式更改为读取为右摇杆上的输入。方向输入始终是该轴上的最小值、最大值或零值。

**默认**: <Hotkey buttons={["S1", "S2", "Right"]}/>

## Home 键

此快捷键模拟 <Hotkey buttons={["A1"]}/> 键的按压，因为并非所有控制器本身都有此键。

**默认**: <Hotkey buttons={["S1", "S2", "Up"]}/>

## Capture 键

此快捷键模拟 <Hotkey buttons={["A2"]}/> 键的按压，因为并非所有控制器本身都有此键。

## SOCD 上优先

此快捷键将 SOCD 覆盖方法更改为解决为 X 轴上的中性输入（无输入），但在同时按下两个方向时优先考虑 Y 轴上的 `上` 输入。

**默认**: <Hotkey buttons={["S2", "A1", "Up"]}/>

| 第一输入 + 第二输入 | 结果  |
| :-----------------: | :---: |
|     左 + 右         | 回中  |
|     右 + 左         | 回中  |
|     上 + 下         |  上   |
|     下 + 上         |  上   |

## SOCD 回中

此快捷键将 SOCD 覆盖方法更改为解决为 X 轴和 Y 轴上的回中输入（无输入），当同时按下两个方向时。

**默认**: <Hotkey buttons={["S2", "A1", "Down"]}/>

| 第一输入 + 第二输入 | 结果  |
| :-----------------: | :---: |
|     左 + 右         | 回中  |
|     右 + 左         | 回中  |
|     上 + 下         | 回中  |
|     下 + 上         | 回中  |

## SOCD 后输入优先

此快捷键将 SOCD 覆盖方法更改为在同时按下两个方向时优先考虑 X 轴和 Y 轴上的第二个方向输入。

**默认**: <Hotkey buttons={["S2", "A1", "Left"]}/>

| 第一输入 + 第二输入 | 结果  |
| :-----------------: | :---: |
|     左 + 右         |  右   |
|     右 + 左         |  左   |
|     上 + 下         |  下   |
|     下 + 上         |  上   |

## SOCD 先输入优先

此快捷键将 SOCD 覆盖方法更改为在同时按下两个方向时优先考虑 X 轴和 Y 轴上的第一个方向输入。

| 第一输入 + 第二输入 | 结果  |
| :-----------------: | :---: |
|     左 + 右         |  左   |
|     右 + 左         |  右   |
|     上 + 下         |  上   |
|     下 + 上         |  下   |

## SOCD 关闭

此快捷键将 SOCD Cleaning 方法更改为不在同时按下两个方向时解决 X 轴和 Y 轴上的任何方向输入。控制器将发送两个方向输入。

:::note

此覆盖模式仅在 XInput 模式下可用，因为 PS3/DirectInput、PS4 和 Nintendo Switch 模式不支持设置 SOCD 关闭，会默认设为中性 SOCD 覆盖模式。

:::

| 第一输入 + 第二输入 |      结果      |
| :-----------------: | :------------: |
|     左 + 右         | 左和右        |
|     右 + 左         | 左和右        |
|     上 + 下         | 上和下        |
|     下 + 上         | 上和下        |

## 反转 X 轴

此快捷键将反转控制器的 X 轴（即按右键将输入左，反之亦然）。

## 反转 Y 轴

此快捷键将反转控制器的 Y 轴（即按上键将输入下，反之亦然）。

## 切换四向操纵杆模式

此快捷键会防止非主方向的注册。当接收到第二个主方向输入时，游戏手柄方向将更改为最新的主方向，类似于 [SOCD 后输入优先](#socd-后输入优先)。

## 切换 DDI 四向操纵杆模式

此快捷键将改变与 [双方向输入](./add-ons/dual-direction-input.mdx) 插件映射的方向键的行为，以防止非主方向的注册。当接收到第二个主方向输入时，游戏手柄方向将更改为最新的主方向，类似于 [SOCD 后者优先](#socd-后输入优先)。

## 加载档案 #1-4

此快捷键将加载在网页配置器中的 [档案设置](./web-configurator/menu-pages/02-gpio-pin-mapping.mdx#档案) 中设置的各种输入到 GPIO 引脚映射档案。

:::caution 多个加载档案快捷键

建议配置多个`加载档案`快捷键，以防您更改档案后无法更改回去。

:::

## 下一个档案

此快捷键循环通过在网页配置器中的 [档案设置](./web-configurator/menu-pages/02-gpio-pin-mapping.mdx#档案) 中设置的 GPIO 引脚映射档案。
例如，如果`Profile 1`是活动的，按下快捷键将循环到下一个启用的档案，例如 `Profile 2`。到达最后一个档案后，将循环回第一个档案。

## 上一个档案

此快捷键循环通过在网页配置器中的 [档案设置](./web-configurator/menu-pages/02-gpio-pin-mapping.mdx#档案) 中设置的 GPIO 引脚映射档案。
例如，如果`Profile 2`是活动的，按下快捷键将循环到上一个启用的档案，例如`Profile 1`。到达第一个档案后，将循环回最后一个档案。

## L3 键

此快捷键模拟 <Hotkey buttons={["L3"]}/> 键的按压，因为并非所有控制器本身都有此键。

## R3 键

此快捷键模拟 <Hotkey buttons={["R3"]}/> 键的按压，因为并非所有控制器本身都有此键。

## 触摸板

此快捷键模拟触摸板的按压，因为并非所有控制器本身都有此键。

:::note

触摸板与 [A1 键](#a1-键) 快捷键不同，因为此快捷键无论 `切换触摸板和共享` 开关都会触发触摸板。

:::

## 重启 GP2040-CE

此快捷键将在无需按下板上的硬件重置按钮（如果存在）或者拔掉控制器与 PC 或游戏主机连接的情况下重启控制器。
