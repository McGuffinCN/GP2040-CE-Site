# PCF8575 IO 扩展器

用途：此附加功能允许您使用基于 PCF8575 的 IO 扩展器。

![GP2040-CE 配置器 - 附加功能 DRV8833 震动](../assets/images/gpc-add-ons-pcf85750io-expander.png)

## Web 配置器选项

对于 16 个可用的扩展器引脚中的每一个，您可以将其设置为输入或输出引脚，并选择与之绑定的按钮。

### 要求

此附加功能需要一个基于 PCF8575 的 IO 扩展器。
