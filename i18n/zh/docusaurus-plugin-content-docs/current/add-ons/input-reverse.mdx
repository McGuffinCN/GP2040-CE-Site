# 输入反转

用途：此插件允许用户通过按下或按住一个额外的按钮，细化地反转、置于中立或保持游戏手柄上的基本方向不变，并输入新的方向。

![GP2040-CE 配置器 - 插件 输入反转](../assets/images/gpc-add-ons-input-reverse.png)

## 网页配置器选项

- `反转输入引脚` - 用于反转输入按钮的 GPIO 引脚。
- `反转输入引脚 LED` - 用于反转输入 LED 的 GPIO 引脚。
- `反转上` - 当按下或按住反转输入按钮时，启用上方向反转。
- `反转下` - 当按下或按住反转输入按钮时，启用下方向反转。
- `反转左` - 当按下或按住反转输入按钮时，启用左方向反转。
- `反转右` - 当按下或按住反转输入按钮时，启用右方向反转。

四个方向中的每一个都可以独立设置以下三种行为之一，当按住反转输入引脚时生效：

- `禁用` - 此方向不发生变化（例如，上 + 反转 = 上）。
- `启用` - 此方向被反向激活（例如，左 + 反转 = 右）。
- `中立` - 此方向被置于中立（例如，下 + 反转 = 中立）。

## 硬件

### 要求

此插件需要一个额外的按钮映射到 GPIO 引脚。

### 安装

对于街机按钮和键盘开关，两侧的触点需要分别连接到 GPIO 引脚（例如，通过导线或 PCB 上的走线）和接地（例如，通过导线或 PCB 上的走线）。
