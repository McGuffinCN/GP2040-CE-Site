# I2C 模拟 ADS1219

用途：此插件旨在使用 ADS1219 24 位模拟-数字转换器（ADC）添加左右模拟摇杆功能。

![GP2040-CE 配置器 - 插件 ADS1219](../assets/images/gpc-add-ons-ads1219.png)

## 网页配置器选项

:::info GPIO 引脚分配

I2C 的 SDA 和 SCL 引脚以及速度现在可以在 [`配置 > 外围设备映射 - I2C`](../web-configurator/menu-pages/03-peripheral-mapping.mdx#i2c) 中进行配置。

:::

- `I2C 模拟 ADS1219 块` - 要使用的 I2C 块（i2c0 或 i2c1）。
- `I2C 模拟 ADS1219 地址` - 根据 SDA 和 SCL 引脚的使用情况设置 I2C 模拟 ADS1219 的地址，详见以下表格。

| A0  | A1  | I2C 地址   |
| :-: | --- | :---------: |
| SDA | SCL |   1001110   |
| SCL | SDA |   1001011   |

## 硬件

### 要求

此插件需要 ADS1219 模拟-数字转换器（ADC）才能正常工作。

### 安装

根据以下表格连接 ADS1219 的引脚：

| ADS1219 引脚 | 功能                     |
| :----------: | ------------------------ |
|     A0       | SDA 或 SCL               |
|     A1       | SDA 或 SCL               |
|    AIN0      | 左模拟摇杆 X 轴          |
|    AIN1      | 左模拟摇杆 Y 轴          |
|    AIN2      | 右模拟摇杆 X 轴          |
|    AIN3      | 右模拟摇杆 Y 轴          |

<br />

:::info A0/A1 GPIO 分配

这些引脚需要连接到 RP2040 上的适当 SDA/SCL 引脚，并在 网页配置器中设置正确的 I2C 块。

有关设置 I2C 的更多信息，请参阅 [外围设备映射](../web-configurator/menu-pages/03-peripheral-mapping.mdx#i2c)。

:::

## 其他注意事项

这些安装说明仅涵盖 ADS1219 ADC 的连接，不包括使用 ADS1219 ADC 所需的额外电路。
