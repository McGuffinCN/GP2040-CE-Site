# 热键指南

选择按钮标签以在使用指南中显示：

<InputLabelSelector />
<br />

:::note

所有以下热键绑定都可以在 Web 配置器的[热键设置](./web-configurator/menu-pages/01-settings.mdx#hotkey-settings)中更改或分配为不同的按钮组合。

:::

## 无操作

此热键无功能。使用这个热键来禁用特定的热键组合。

## 十字键模式

此热键将 DPad 模式更改为读取为十字键上的数字输入。

**默认**: <Hotkey buttons={["S1", "S2", "Down"]}/>

## 左摇杆模式

此热键将 DPad 模式更改为读取为左模拟操纵杆上的输入。方向输入始终是该轴上的最小值、最大值或零值。

**默认**: <Hotkey buttons={["S1", "S2", "Left"]}/>

## 右摇杆模式

此热键将 DPad 模式更改为读取为右模拟操纵杆上的输入。方向输入始终是该轴上的最小值、最大值或零值。

**默认**: <Hotkey buttons={["S1", "S2", "Right"]}/>

## Home 按钮

此热键模拟 <Hotkey buttons={["A1"]}/> 按钮的按压，因为并非所有控制器本身都有此按钮。

**默认**: <Hotkey buttons={["S1", "S2", "Up"]}/>

## Capture 按钮

此热键模拟 <Hotkey buttons={["A2"]}/> 按钮的按压，因为并非所有控制器本身都有此按钮。

## SOCD 上优先

此热键将 SOCD 清理方法更改为解决为 X 轴上的中性输入（无输入），但在同时按下两个方向时优先考虑 Y 轴上的 `上` 输入。

**默认**: <Hotkey buttons={["S2", "A1", "Up"]}/>

| 第一输入 + 第二输入 | 结果  |
| :-----------------: | :---: |
|     左 + 右         | 回中  |
|     右 + 左         | 回中  |
|     上 + 下         |  上   |
|     下 + 上         |  上   |

## SOCD 回中模式

此热键将 SOCD 清理方法更改为解决为 X 轴和 Y 轴上的回中输入（无输入），当同时按下两个方向时。

**默认**: <Hotkey buttons={["S2", "A1", "Down"]}/>

| 第一输入 + 第二输入 | 结果  |
| :-----------------: | :---: |
|     左 + 右         | 回中  |
|     右 + 左         | 回中  |
|     上 + 下         | 回中  |
|     下 + 上         | 回中  |

## SOCD 后输入优先

此热键将 SOCD 清理方法更改为在同时按下两个方向时优先考虑 X 轴和 Y 轴上的第二个方向输入。

**默认**: <Hotkey buttons={["S2", "A1", "Left"]}/>

| 第一输入 + 第二输入 | 结果  |
| :-----------------: | :---: |
|     左 + 右         |  右   |
|     右 + 左         |  左   |
|     上 + 下         |  下   |
|     下 + 上         |  上   |

## SOCD 先输入优先

此热键将 SOCD 清理方法更改为在同时按下两个方向时优先考虑 X 轴和 Y 轴上的第一个方向输入。

| 第一输入 + 第二输入 | 结果  |
| :-----------------: | :---: |
|     左 + 右         |  左   |
|     右 + 左         |  右   |
|     上 + 下         |  上   |
|     下 + 上         |  下   |

## SOCD 关闭

此热键将 SOCD Cleaning 方法更改为不在同时按下两个方向时解决 X 轴和 Y 轴上的任何方向输入。控制器将发送两个方向输入。

:::note

此清理模式仅在 XInput 模式下可用，因为 PS3/DirectInput、PS4 和 Nintendo Switch 模式不支持设置 SOCD 清理关闭，会默认设为中性 SOCD 清理模式。

:::

| 第一输入 + 第二输入 |      结果      |
| :-----------------: | :------------: |
|     左 + 右         | 左和右        |
|     右 + 左         | 左和右        |
|     上 + 下         | 上和下        |
|     下 + 上         | 上和下        |

## 反转 X 轴

此热键将反转控制器的 X 轴（即按下右按钮将导致左输入，反之亦然）。

## 反转 Y 轴

此热键将反转控制器的 Y 轴（即按下上按钮将导致下输入，反之亦然）。

## 切换四向操纵杆模式

此热键会防止非主方向的注册。当接收到第二个主方向输入时，游戏手柄方向将更改为最新的主方向，类似于[SOCD 后者优先](#socd-last-win)。

## 切换 DDI 四向操纵杆模式

此热键将改变与[双方向输入](./add-ons/dual-direction-input.mdx)插件映射的方向按钮的行为，以防止非主方向的注册。当接收到第二个主方向输入时，游戏手柄方向将更改为最新的主方向，类似于[SOCD 后者优先](#socd-last-win)。

## 加载配置文件 #1-4

此热键将加载在 Web 配置器中的[配置文件设置](./web-configurator/menu-pages/02-gpio-pin-mapping.mdx#profiles)中设置的各种输入到 GPIO 引脚映射配置文件。

:::caution 多个加载配置文件热键

建议配置多个`加载配置文件`热键，以防您更改配置文件后无法更改回去。

:::

## 下一个配置文件

此热键循环通过在 Web 配置器中的[配置文件设置](./web-configurator/menu-pages/02-gpio-pin-mapping.mdx#profiles)中设置的 GPIO 引脚映射配置文件。
例如，如果`配置文件1`是活动的，按下热键将循环到下一个启用的配置文件，例如`配置文件2`。到达最后一个配置文件后，将循环回第一个配置文件。

## 上一个配置文件

此热键循环通过在 Web 配置器中的[配置文件设置](./web-configurator/menu-pages/02-gpio-pin-mapping.mdx#profiles)中设置的 GPIO 引脚映射配置文件。
例如，如果`配置文件2`是活动的，按下热键将循环到上一个启用的配置文件，例如`配置文件1`。到达第一个配置文件后，将循环回最后一个配置文件。

:::note

触摸板按钮与[A1按钮](#a1-button)热键不同，因为此热键在`切换触摸板和共享`开关启用或禁用时都会触发触摸板按钮。

:::

## 重启 GP2040-CE

此热键将在无需按下板上的硬件重置按钮（如果存在）或者拔掉控制器与 PC 或游戏主机连接的情况下重启控制器。
