# 数据备份与恢复

![GP2040-CE 配置器 - 附加功能备份与恢复](../../assets/images/gpc-backup-and-restore.png)

- `备份到文件` - 允许您选择备份哪些内容到文件（默认情况下全部选中）。
- `从文件恢复` - 允许您选择从文件恢复哪些内容（默认情况下全部选中）。

:::warning

从 GP2040-CE 版本 0.7.6 开始，导入早期版本备份中的 GPIO 引脚映射将无效。如果您的控制器固件文件可用，建议对控制器进行 [强制清除刷写](../../installation.mdx#flash-nuke-process) 并重新刷写 GP2040-CE v0.7.6。

请注意，这将清除控制器在强制清除刷写过程之前的所有数据。

:::
