# 网页配置器 - 主页

您可以在此查看设备的系统状态。

-----

## 版本

`Device Name` (设备名称) - 这将告诉您设备的名称以及上传到设备上的UF2文件的名称。
`Current` (当前版本) - 设备上正在运行的固件的当前版本。
`Latest` (最新版本) - 在我们的发布区中可用的最新固件版本。

-----

## 内存

本节显示了有关闪存模块的各种状态。

:::note

GP2040-CE要求设备至少具有2MB闪存（这是树莓派Pico的标准配置）。

有少量报告称，来自速卖通（AliExpress）等地方的一些低成本克隆设备只配备了1MB闪存。

应避免使用只有1MB闪存的设备，因为它们可能无法工作或无法按预期工作。

如果您正在制作自己的设备，请确保使用至少2MB的闪存模块。

:::
