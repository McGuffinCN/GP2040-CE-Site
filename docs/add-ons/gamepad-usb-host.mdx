# 游戏手柄 USB 主机配置

用途：此附加功能旨在允许游戏手柄与 GP2040-CE 一起使用。例如，可以将受支持的 USB 控制器插入可用的 USB 主机端口，然后使用该 USB 控制器进行游戏。此功能可以将受支持的控制器转换为适用于其原本不支持的系统。

![GP2040-CE 配置器 - 游戏手柄 USB 主机](../assets/images/gpc-add-ons-keyboard-gamepad-usb-host.png)

## Web 配置器选项

目前，您只能启用或禁用此附加功能。

请注意，目前您必须在 `键盘主机` 和 `游戏手柄 USB 主机` 之间进行选择。

## 硬件

要使用此附加功能，您需要以下之一的游戏手柄：
- Sony Dualshock 4 控制器
- Google Stadia 控制器
- Ultrastick 360（2015 年之前版本）
- Ultrastick 360

### 要求

:::note

请注意，未来可能会增加更多控制器的支持。目前我们不接受添加额外控制器的请求。

:::
