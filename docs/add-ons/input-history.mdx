# 输入历史

用途：此附加功能旨在允许您显示输入历史。

![GP2040-CE 配置器 - 附加功能 输入历史](../assets/images/gpc-add-ons-input-history.png)

## Web 配置器选项

- `历史长度` - 设置历史记录的最大字符长度。
- `列` - 指定历史记录在显示屏上开始的列位置。（0 - 左侧，20 - 右侧）
- `行` - 指定历史记录在显示屏上出现的行位置。（0 - 顶部，7 - 底部）

## 硬件

### 要求

已配置的 [单色显示屏](../web-configurator/menu-pages/07-display-configuration.mdx)。

### 安装

无需安装。
