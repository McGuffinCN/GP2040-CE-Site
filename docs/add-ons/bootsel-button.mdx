# BOOTSEL 按钮配置

用途：此附加功能旨在允许将 BOOTSEL 按钮作为额外的按钮输入使用。

![GP2040-CE 配置器 - 附加功能 BOOTSEL 按钮配置](../assets/images/gpc-add-ons-bootsel-button.png)

## Web 配置器选项

- `BOOTSEL Button` - 选择一个输入映射到 BOOTSEL 按钮。

## 硬件

### 要求

此附加功能仅适用于具有 BOOTSEL 按钮的设备。

### 安装

此附加功能没有安装说明。

## 其他注意事项

如果设置了此按钮，OLED 可能会变得无响应。您可以取消设置以恢复 OLED 功能。

:::warning

`BOOTSEL 按钮配置` 仅适合有经验的用户使用。

:::
