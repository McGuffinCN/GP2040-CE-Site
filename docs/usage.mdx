## Bootsel 模式

Bootsel 模式是板进入可刷入固件的状态。每当进行更新或排查问题时，您都需要使用此模式。

您可以通过以下方式进入 Bootsel 模式：在插入控制器时按住 <Hotkey buttons={["S1", "S2", "Up"]}/> 按钮，通过进入 Web 配置器并在 Bootsel 模式下重启，或在控制器插入状态下按住 <Hotkey buttons={["S1", "B3", "B4"]}/> 五秒钟。激活后，除了刷入新固件或拔掉设备，没有其他方法可以离开 Bootsel 模式。

通常，板上会有一个 Bootsel 按钮。这个按钮往往是 PCB 上露出的按钮，或者在控制器内部，需要针状物按压或拆卸控制器才能按到。

:::note

一些控制器选择不包含 Bootsel 按钮。在这种情况下，您需要找到按钮的位置，并在插入或启动控制器时使用导体连接触点或垫片的两侧。

:::

## WebConfig 模式

WebConfig 模式是板启动内置网页浏览器配置应用程序的状态。在此，您可以根据需要自定义和配置控制器。更多信息请点击[这里](./web-configurator/web-configurator.mdx)。

您可以通过以下方式进入 WebConfig 模式：在插入控制器时按住 <Hotkey buttons={["S2"]}/> 按钮，或在控制器插入状态下按住 <Hotkey buttons={["S2", "B3", "B4"]}/> 五秒钟。再次按住这个组合五秒钟，即可将设备切换回先前使用的控制器模式。

## 输入模式

GP2040-CE兼容多个系统和输入模式。可以在Web Configurator中设置按住八个按钮时分配的特定输入模式，配置路径为[设置](./web-configurator/menu-pages/01-settings.mdx#boot-input-modes)。

不过，预编译固件中有默认输入模式。要更改输入模式，**在插入控制器时按住以下按钮之一：**

| 主输入模式      |        按住的按钮         |
| :------------- | :-----------------------: |
| Nintendo Switch | <Hotkey buttons={["B1"]}/> |
| XInput          | <Hotkey buttons={["B2"]}/> |
| PS3/DirectInput | <Hotkey buttons={["B3"]}/> |
| PS5             | <Hotkey buttons={["B4"]}/> |
| Xbox One        | <Hotkey buttons={["R1"]}/> |
| 键盘            | <Hotkey buttons={["R2"]}/> |
| Original XBox   |      `未分配`            |

| Mini/Classic 游戏机          | 按住的按钮 |
| :--------------------------- | :--------: |
| Sega Genesis/MegaDrive Mini  | `未分配`   |
| NEOGEO mini                  | `未分配`   |
| PC Engine/Turbografx 16 Mini | `未分配`   |
| EGRET II Mini                | `未分配`   |
| ASTROCITY Mini               | `未分配`   |
| Playstation Classic          | `未分配`   |

:::note

输入模式在重启后会被保留。

:::

## 快捷键

GP2040设有多个快捷键，无需访问Web Configurator即可更改功能。所有快捷键和快捷键绑定可以在Web Configurator的[快捷键设置](./web-configurator/menu-pages/01-settings.mdx#hotkey-settings)中更改或分配为不同的按钮组合。

常用快捷键包括将方向更改为模拟数字DPad、左模拟摇杆、右模拟摇杆或更改[SOCD（同时相反方向）](https://glossary.infil.net/?t=SOCD)清理模式。不同的SOCD清理模式将确保控制器在发送输入时遵循特定的方向输入规则。

许多快捷键默认启用，如果控制器出现意外行为，请确保未意外触发快捷键。

| 默认启用的快捷键                           | 输入                              |
| :----------------------------------------- | :-------------------------------- |
| [主页按钮](hotkeys.mdx#home-button)        | <Hotkey buttons={["S1","S2","Up"]}/> |
| [DPad 数字](hotkeys.mdx#dpad-digital)      | <Hotkey buttons={["S1","S2","Down"]}/> |
| [DPad 左模拟](hotkeys.mdx#dpad-left-analog) | <Hotkey buttons={["S1","S2","Left"]}/> |
| [DPad 右模拟](hotkeys.mdx#dpad-right-analog) | <Hotkey buttons={["S1","S2","Right"]}/> |
| [SOCD 上优先](hotkeys.mdx#socd-up-priority) | <Hotkey buttons={["S2","A1","Up"]}/>    |
| [SOCD 中立](hotkeys.mdx#socd-neutral)     | <Hotkey buttons={["S2","A1","Down"]}/>  |
| [SOCD 最后的胜出](hotkeys.mdx#socd-last-win) | <Hotkey buttons={["S2","A1","Left"]}/>  |

:::note

选定的DPad模式和SOCD清理模式在重启后会被保留。

:::
